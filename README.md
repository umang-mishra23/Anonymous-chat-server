# Anonymous Chat Server

## Run
```bash
npm install
node server.js
```

Open `http://localhost:3000`.

> GIF and sticker search requires a GIPHY API key in `GIPHY_API_KEY` (or `API_KEY`) before starting the server.

## Included
- Public and private tabs
- Anonymous profiles with avatar + temporary display name
- Public-only private chat request card
- Accept / deny flow
- Auto private room shift on accept
- Typing indicators
- Reply system
- Emoji reactions
- GIF / image URL previews
- Theme selector
- Sound alerts toggle
- RAM-only state
